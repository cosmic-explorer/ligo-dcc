DCC version 2.2.2 was released on October 8, 2009.

In addition to a few bug fixes this DCC release adds the following functionality. If you have any questions please contact dcc-help@ligo.org.

The default behavior of author searches now excludes matches against author groups such as "LSC" since most people are interested in searching for "small author list" documents. It is still possible to match against author groups in the Advanced Search by selecting Include Author Groups halfway down the page.

The simple Text Search was changed to perform a more restrictive search for documents that include ALL the specified strings rather than the previous more inclusive match for documents that contain ANY of the specified strings. The previous behavior is still available from the Advanced Search.

The DCC search engine has been upgraded to a significantly faster system that will scale to a significantly larger number of documents and is synchronously updated when new documents are uploaded, i.e., it is no longer necessary to wait for up to 30 minutes for new content to be indexed.

The document group access structure has been extended to allow sub-groups to be created relative to the main group hierarchy that inherit membership from a parent group. The initial example of this is a new AdLIGO_Business group which has implicit access rights for members of its parent Lab_Business group.

Documents that are publicly accessible, i.e., do not require an account to access, now have their public address listed as the most prominent link when viewing such a document. This is to facilitate sharing URLs with external collaborators. For example, when viewing the October 2008 Author List from a logged in session you will now see in the upper left corner a Public link that anyone this is sent to can access without having to login. The internal Private link: is still exposed below the Public link: for circulation to collaboration colleagues who might want to modify a document you direct them to.

Please also note that most internal scientific discussions should be marked for viewing by the group LVC to share with both the LSC and Virgo collaborations unless it really is private content that should be restricted to a smaller group. For example, many G- presentation documents during the last L-V meeting where mistakenly left as viewable by only the group LSC.
