DCC Admin[edit]

While the DCC is a self-serving database system, the DCC Administrator is always available to accommodate any submittal requests as needed. Just email the DCC at dcc-help@ligo.org with the minimum pertinent information (document type, author(s), title, brief abstract, topic & the viewing/modifying group) along with the actual document and we will gladly upload it to the DCC database on your behalf.

Viewing PDF files Properly in the Firefox Browser[edit]

The recent Firefox update (version 19.0.2) defaults to a built-in, non-Adobe reader for PDF files. This default PDF reader doesn't show red-lines added to Adobe pdf files and (more importantly) screws up fonts used in SolidWorks resulting in unreadable drawings. In order to correct this, do the following to switch from the built-in PDF viewer:

  1. At the top of the Firefox window, click on the Firefox button and select Options
  2. Select the Applications panel
  3. Find "Portable Document Format (PDF)" in the list and click on it to select it.
  4. Click on the drop-down arrow in the "Action" column for the above entry and select the PDF viewer you wish to use, e.g. "Use Adobe Acrobat (in Firefox)"
